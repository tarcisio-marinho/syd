# syd
Github commit/clonning automatization tool

# Compiling 

    gcc syd.c -o syd
    
    g++ clone.cpp -o clone

    
# Terminal Comand

    sudo cp syd /usr/bin/
    sudo cp pull.sh /usr/bin/pull
    sudo cp clone /usr/bin/clone

# USAGE

    SYD:
        
        syd [nome do commit]
    
    CLONE:
        
        clone [URL]
        clone [USER] [REPO]
        clone [REPO] # clone from tarcisio-marinho # my own profile

    PULL:

        pull
